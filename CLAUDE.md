# Contexto del proyecto — Trackion

Trackion es un sistema de mesa de ayuda (helpdesk) serverless en Python sobre AWS, con PostgreSQL
y un módulo de integración de APIs extensible. Flujo de trabajo: **Spec-Driven Development (OpenSpec)**.

Lee y aplica:
- @docs/base-standards.md — principios base, idioma, planificación, reglas OpenSpec.
- @docs/backend-standards.md — backend serverless Python (capas, datos, integraciones).
- @docs/frontend-standards.md — frontend vanilla y paleta metálica.
- @docs/ci-cd-standards.md — regla spec-first y pipeline GitHub → AWS.
- @openspec/project.md — contexto del proyecto.

Regla dura: **spec-first** (nada de código antes de su spec). Ver README.md para el flujo `/opsx:*`.
